# Crowned Reads 👑⚔️

An interactive blog website for young adult romantasy readers.  
Final project for **YBS226 - Advanced Web Design** at Piri Reis University.

## About the Project
Crowned Reads is a community-style blog where readers of fantasy and romantasy 
novels can share their thoughts on popular books and tag them based on their 
themes (love triangles, enemies-to-lovers, magic systems, etc.).

## Featured Books
- **Caraval Series** — Stephanie Garber  
- **Once Upon a Broken Heart Series** — Stephanie Garber  
- **Fourth Wing** — Rebecca Yarros  
- **A Court of Thorns and Roses (ACOTAR)** — Sarah J. Maas  

## Technologies Used
- **HTML5** — semantic page structure  
- **SCSS / CSS3** — styling with variables, partials and nesting  
- **JavaScript (ES6)** — interactivity (comments, tag filtering, navigation)  
- **Live Sass Compiler** — SCSS → CSS compilation  
- **Prettier** — code formatting  
- **Git & GitHub** — version control
